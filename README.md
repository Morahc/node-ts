# Image processing API

## Scripts
- Install: ```npm install```
- Build: ```npm run build```
- Lint: ```npm run lint```
- Prettier: ```npm run format```
- Run unit tests: ```npm test```
- Start server:development: ```npm run dev```
- Start server:production: ```npm run start```

#### Image endpoint
http://localhost:3000/api/image?filename=fjord

#### Thumbnail processing endpoint
http://localhost:3000/api/image?filename=fjord&width=200&height=200
